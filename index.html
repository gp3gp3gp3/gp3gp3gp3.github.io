A roadside assistance community web application for cyclists

Technologies: PostgresDB, Ruby on Rails, Javascript, JQuery, HTML5, CSS3, Foundation, GoogleMaps API, Twillio API

Similar to triple a, but for cyclists. This web application was built with mobile first design. A user can put up an alert when they've broken down and are in need of assistance. Users can also help other others in need of assistance if they have spare parts on them. The application sends out texts to both parties when the mechanic agrees to meet the broken down user. The text will have the mechanic's phone number and the amount of time it will take to get there on a bicycle.
